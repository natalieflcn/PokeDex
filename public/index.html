How to Use Naty's Pokédex

Browse Pokémon, view detailed stats, track which ones you’ve caught, and mark your favorites.

Track where Pokémon were caught using an interactive map. Note: Location data is never stored externally.

View your caught and favorite Pokémon within my profile dashboard — your activity lives inside my Pokédex.

Meet Naty

My name is Natalie (Naty) and I’m a full-stack developer who’s honing her skills by building clean, interactive, and meaningful web applications.

Why I Built Naty's Pokédex

This is the first major project in my coding portfolio. I wanted to challenge myself by building a fully functional application using vanilla JavaScript (no frameworks) to strengthen my understanding of core fundamentals.

Growing up playing Pokémon games, this felt like the perfect way to merge a nostalgic passion with technical growth.

What I Learned from this Project

Version 2.0 of Naty's Pokédex demonstrates significant growth –– both in the original codebase and in my developer journey.


Refactoring Version 1.0 allowed me to:

  • Strengthen my implementation of MVC architecture
  • Improve state management patterns
  • Handle asynchronous programming and race conditions
  • Build modular, maintable, and scalable code

This project reflects not just technical ability, but it is the product of discipline, persistence, and continous improvement.
